Mobile Navigation Demo

This page demonstrates how to use the mobile navigation functionality. Try clicking on the hamburger menu in the top-right corner to see the mobile navigation in action.

How to Add Mobile Navigation to Your Site

To add the mobile navigation functionality to your site, follow these steps:

  1. Include the styles.css file in your HTML:
    2. <link rel="stylesheet" href="styles.css">
  2. Add the following HTML structure for your navigation:
    3. <header class="header">
    <div class="nav-container">
        <a href="#" class="logo"> 
            <img src="logo1.png" width="160" height="80" alt="Logo">
        </a>
        <div class="hamburger">
            <div class="hamburger-line"></div>
            <div class="hamburger-line"></div>
            <div class="hamburger-line"></div>
        </div>
        <nav>
            <ul class="nav-menu">
                <li><a href="#">+ Home</a></li>
                <li><a href="#">+ Rooms</a></li>
                <li><a href="#">+ Gallery</a></li>
                <li><a href="#">+ About US</a></li>
                <li><a href="#">+ Contact</a></li>
            </ul>
        </nav>
    </div>
</header>
  3. Include the nav-loader.js script at the end of your HTML, just before the closing body tag:
    4. <script src="nav-loader.js"></script>

Note

The nav-loader.js script automatically loads the mobile-nav.js file, which contains the JavaScript code for the mobile navigation functionality.
